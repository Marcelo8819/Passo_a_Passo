HTML Básico

O html não é uma linguagem de programção , mas serve como um modelo de estruturação para websites, na verdade a linguagem de programação que faz as coisas acontecem é o JavaScript, dando maior interatividade a pagina, sem ele tudo ficaria paradinho ou melhor dizendo estático.

Vamos falar sobre:

As coisas em html giram em torno de tags, elas são esses sinais de menor < seguida de uma palavra chave, e por último o sinal de maior > , pode se acostumar por você irá usar elas inúmeras vezes. Por exempro o botão aqui embaixo "Criar Botão" usa tags , assim como o play , que redireciona para outra página.





Entradas de dados , o tão falado input

Em html nos usamos um elemento chamado input para coletar dados, tipo o nome de usuário, email , senhas entre outros. Para criar um input usaremos as tag < > mais a palavra chave input.O elemento placeholder indica na caixa o dado a ser inserido.
